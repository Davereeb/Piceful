Meet Piceful

Your AI photo guide.

Piceful is a browser-based AI photography assistant that helps anyone recreate a reference photo in real time. Upload a target image, and on-device AI detects your body position through the camera, compares it to the reference, and gives you live guidance until you match the framing.

No app install. Works on any smartphone. Combines on-device pose detection with a cloud vision model for natural-language coaching.

How It Works Tech Stack

On-device AI meets
cloud vision.

Layer 01

TensorFlow.js

BlazeFace for face detection and MoveNet for 17-keypoint body pose estimation. Runs on-device at ~10fps with zero server cost. Also powers our Premium skeleton overlay — color-coded dots on the reference image turn green as you match the pose.

On-device~10fps
Layer 02

Qwen-VL-Max

Alibaba’s vision-language model via DashScope API. Analyzes reference + live frame pairs and generates natural-language coaching instructions.

Cloud AIMultimodal
Layer 03

WebRTC

getUserMedia for direct camera access in the browser. Supports front and back camera switching with no native app required.

Browser APINo install
Layer 04

Supabase

PostgreSQL database with Row Level Security for user auth, cloud collections storage, and image hosting.

BackendAuth + Storage
